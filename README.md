# CellOS

The first operating system I wrote in C.
It is a collection of different Cellular Automatas

# Compiling

## Requirements:

- gcc-multilib
- binutils
- nasm
- grub-pc-bin
- qemu *(Optional for running)*

## Compiling:

```bash
make
```

## Running:

Either burn to a physical CD or run:
```bash
make run
```
